# inventory-docs

Shared planning and specifications for the **Inventory** product — the single source of truth for everything that spans both the Android app and the Laravel backend.

## What lives here

| Path         | Purpose                                                                       |
| ------------ | ----------------------------------------------------------------------------- |
| `planning/`  | Raw product planning and the breakdown that splits work into Android/Laravel. |
| `specs/`     | Cross-cutting specs: product vision, data model / ERD, and the API contract.  |

App-specific planning lives in each application repo's own `docs/` directory.

## Related repositories

- [`inventory-android`](https://github.com/spdotdev/inventory-android) — Android client
- [`inventory-laravel`](https://github.com/spdotdev/inventory-laravel) — Laravel backend / API
